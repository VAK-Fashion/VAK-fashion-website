## VAK-fashion-website

# Download
```bash
git clone https://github.com/VAK-Fashion/VAK-fashion-website.git
```
**command for linux user**
```bash
#for backende server
cd backend/src/
python manage.py runserver

```
```bash
cd frontend/app
#this for install pack
npm i 
#this for frontend sever
npm run dev
```

CMD

```cmd
python ./manage.py runserver
```
```cmd
cd frontend/app

npm i 

npm run dev
```
